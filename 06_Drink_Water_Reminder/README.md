# Water Reminder (Python Notification Script)

A simple desktop notification script built with Python to remind users to drink water every 15 minutes.

This project demonstrates scheduling logic, notification handling, and structured scripting.

---

## Features

- Sends desktop notification every 15 minutes
- Console log with timestamp
- Clean exit using CTRL + C
- Beginner-friendly structured code

---

## Tech Stack

- Python 3.x
- plyer (for desktop notifications)
- time module
- datetime module

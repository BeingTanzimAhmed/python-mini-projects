# PDF Merger (Tkinter GUI)

A lightweight desktop application built using Python and Tkinter to merge multiple PDF files into a single document.

This project focuses on GUI fundamentals, file handling, validation logic, and clean error handling.

---

## Features

- Select multiple PDF files
- Prevent duplicate selection
- Remove selected files
- Clear all files
- Scrollable file list
- Merge button enabled only when 2+ PDFs are selected
- Exception handling with user-friendly alerts

---

## Tech Stack

- Python 3.x
- Tkinter (GUI)
- pypdf (PDF processing)
- os (file utilities)

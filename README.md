# Python Mini Projects

This repository contains small Python projects created to practice and apply
core Python concepts through hands-on problem solving.

The goal of this repository is to improve logic building, code organization,
and real-world usage of Python by working on independent mini projects.

## Topics Covered
- Variables and data types
- Conditionals and loops
- Functions
- Object-oriented programming
- File handling
- Logic building

## Projects
- Snake Water Gun
- Quiz Game
- Typing Speed Tester
- File Organizer
- More projects will be added gradually
